
== Setting up

1. download mongodb from mongodb.org

2. install rails 1.9.3

3. install bundler

4. run bundler

   bundle install

5. load db

   mongo backbone_example_development  vendor/assets/javascripts/underscore.js db/seeds.js

6. startup rails

   bundle exec rails s

7. browse to

   http://localhost:3000/

8. Presentation files are in 

   doc/presentations
